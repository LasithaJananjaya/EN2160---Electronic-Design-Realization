
The driver zntport.sys is only mandatory when driving the LPT port under WinNT/XP/2000/7/10.
The driver located in this sub-directory x86 is for 32bits architecture ONLY.
The Install executable will copy zntport.sys in directory Windows/system32/drivers (name in lowercase is mandatory).
If you use the zip package, please perform this copy by yourself.
If no LPT port is detected by my software, please 
  - Close the application 
  - Run the "zntportsetup.exe" software  which is available in the same directory as the application
  - Restart your computer.

IMPORTANT : If your architecture is 64bits, please use the driver zntport.sys located in the x64 sub-directory.
