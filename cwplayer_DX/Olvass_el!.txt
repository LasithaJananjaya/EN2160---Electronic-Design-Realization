 ===========================================
              CW_PLAYER
 ===========================================
       Írta: G. Rivat F6DQM
       http://f6dqm.free.fr
 ===========================================

Ez a szabadon felhasználható (Freeware) program, mely hagyományos morze-kódokat, Q-kódokat, rövidítéseket, véletlen karaktersorozatokat és szövegfájlokat generál.
Ellenorzi a tudásodat egy kvíz tesztjével és segít Neked, hogy jól billentyuzzél a morze jeleid ellenorzésével. Billentyuzésed adás közbeni ellenorzésével a képernyon, elosegíti, hogy kevesebb stresszel és nagyobb biztonsággal vehessél részt a forgalmazásban.
Megfelelo szoftverrel visszajátszható hangfájlokat (*.wav) készít a morzejelekrol.
Saját magad készíthetsz helyi karaktereket.
Egy 20 részes leckesorozaton át tanulhatod meg a morze jeleket. Tartalmukat saját kedved szerint alakíthatod is.

A CW_Player hangkártyát és legalább WinXP-ös (vagy fejlettebb) operációs rendszert igényel.

A Súgóból pár javaslat jelenik meg, ha az egér a kezeloszervek, vagy szövegmezok fölött áll.

Ha Vista alatt nem tudod a súgót megnyitni, kérlek látogass el a
http://www.microsoft.com/downloads/details.aspx?FamilyId=6EBCFAD9D3F5-4365-8070-334CD175D4BB&displaylang=en
oldalra 

A windows másolás (Ctrl+C) és beillesztés (Ctrl+V) parancsai használhatók a visszajátszásnál a véletlen sorozat egy részének kimásolásával és a szövegmezobe illesztésével. Illetve egy fájlba is másolhatod, amit a kedvenc szövegszerkesztoddel nyithatsz meg. (megjegyzem, a szöveg ablak csak egy rekordot fogad, ezért csak egy ASCII fájl másolható ebbe az ablakba.)

Ez a program foként a rádióamator honlapokról töltheto le. Azzal a céllal, hogy segítse a kezdo távírászok és nem rádióamatorök morze tanulását és részükre a rövidhullámok megismerését.

Tehát ne habozz ajánlani ezt a programot a látogatóidnak, ha rádióamator tartalmú honlapod van.

Noha, ezt a programot saját morze tanulásodra használhatod, vagy a morze-oktatód is használhatja, de élo adásban is használható, ha a hangártyád kimenetét az SSB vagy FM adóvevot mikrofon bemenetére csatlakoztatod. 
 
VERY IMPORTANT : 
While running this program under VISTA, WIN7, 8, 10 from a sub-directory under the directory Program Files, 
you will have to start this program as administrator anyway for getting full access to all functions of it, especially the recording of text files, image files, sound files. 
If you don’t do that, the produced files won’t be accessible because of the UserAccountControl (UAC) of Windows.
If you don’t want to run the program as admin, you have to install it in a sub-directory which is NOT under C:\Program Files,
but anyw<here else, for example C:\MyPrograms\. There, the function of the UAC takes no effect.

Version 1.8.2
=============
- The number of choosen characters for random and quiz modes is extended
 to 50. This allows greater character combinations. Characters may be
 repeated several times, that way they will be played more often than
 others.
 For example, the list may include the 26 letters and some of the most
 difficult to copy may be tripled. Then they will be played three times
 more often than the others.

- Back-up and restart of the program with the last "Choosen characters"
 configuration. The back-up file is CWPLAYER.DAT.

- Availability to create your own local characters such as stressed
 vowels.
 Many countries have got such local characters which specific morse
 codes.
 This also allows you to assign dedicated keys to specials characters
 such as SK (end of work) or AS (wait).
 These local characters will played in all modes. When existing, they
 are saved in the back-up file CWPLAYER.DAT.

Version 1.5 features :
======================
- more Q codes

- choice between English and French Language.

The language may be choosen in the command line by typing the following
parameter : /e for English, /f for French. If no parameter, CW_Player 
starts with English.
For example, to start in French, type c:\cw_player.exe /f ,assuming the
executable file is named cw_player.
BE AWARE that under  Win95, when using the Execute command or creating a
short, the command line is displayed between doub le quotes.
Then, the parameter must be typed AFTER the last double quote :
      "C:\CW_Player.EXE" /f
