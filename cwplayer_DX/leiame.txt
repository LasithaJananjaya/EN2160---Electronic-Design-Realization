 =====================================
             CW_Player 
 =====================================
     Escrito por G. RIVAT F6DQM
       http://f6dqm.free.fr
 =====================================

Este programa gratuito gera os caracteres de MORSE comuns, os codigos Q, as abreviaturas
MORSE habituais, series aleatorias, executa ficheiros ASCII, testa os seus conhecimentos
e verifica a sua transmissao manual. 

Depois de se ter treinado a transmitir no vazio, ao verificar
a sua transmissão no monitor vai iniciar o seu primeiro QSO CW com menos stress e mais
confiança.

Este programa gera tambem os ficheiros som (*.wav) de Morse que poderá executar com
qualquer programa de multimedia.

Permite que crie os seus proprios carateres, propondo tambem uma sequência de 20 lições
que poderá modificar a sua vontade.

Para funcionar correctamente necessita de uma placa de som e Windows XP como mínimo.

Alem da ajuda online, alguns balões de texto de ajuda surjem quando o rato se sobrepoe sobre os
comandos principais.

Se nao for possivel abrir o ficheiro de ajuda com a extensão HLP no Windows Vista por favor visita:
http://www.microsoft.com/downloads/details.aspx?FamilyId=6EBCFAD9-D3F5-4365-8070-334CD175D4BB&displaylang=en  

Todas as funções Windows tais como copiar (Control+C) e colar (Control+V) são possíveis.

Principalmente, uma serie aleatória de caracteres pode ser inserida dentro da janela de texto
através do copiar/colar e re-executar ou adicionar dentro de um ficheiro com o vosso editor de texto
favorito para ser executado mais tarde. 

(Note que a janela texto não pode conter mais do que que uma gravação portanto só uma linha termina com retorno de um ficheiro ASCII).

Este programa foi escrito desde o inicio para ser posto à disposição dos cibernautas que visitam sites de Radioamadores.

Tem por finalidade de iniciar na telegrafia e de os ajudar a entrar no nosso Mundo do CW. 

Será também util aos Radioamadores que querem aprender o codigo MORSE para aceder às bandas HF.

Se é administrador de um site WEB, sugira-o aos seus visitantes!

Dito isto, poderá também ser utilizado para o vosso proprio treino, ou pelos instrutores de uma Associação de Radioamadores 
ou mesmo no ar ligando uma das duas saidas da sua placa de som na entrada de modulação do vosso emissor FM ou SSB.

VERY IMPORTANT : 
While running this program under VISTA, WIN7, 8, 10 from a sub-directory under the directory Program Files, 
you will have to start this program as administrator anyway for getting full access to all functions of it, especially the recording of text files, image files, sound files. 
If you don’t do that, the produced files won’t be accessible because of the UserAccountControl (UAC) of Windows.
If you don’t want to run the program as admin, you have to install it in a sub-directory which is NOT under C:\Program Files,
but anyw<here else, for example C:\MyPrograms\. There, the function of the UAC takes no effect.
