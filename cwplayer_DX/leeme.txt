 ===========================================
              CW_PLAYER
 ===========================================
       Escrito por G. Rivat F6DQM
          http://f6dqm.free.fr
 ===========================================

Este programa freeware genera los caracteres del código Morse, el código Q,
las abreviaciones, series de caracteres aleatorios y archivos de texto. 
Pone a prueba su conocimiento por medio de un Quiz y lo ayuda a tener una 
buena manipulación monitoreando su señal de CW. Manipulando sin transmitir 
y verificando su transmisión en pantalla hace luego que su transmision la 
pueda relizar con menos nerviciosismo y mas confianza.
Genera archivos de sonido Wave (*.wav) de caracteres morse los cuales pueden 
ser reproducidos por cualquier programa de multimedios.

Usted puede crear caracteres locales.
Existen 20 lecciones para el aprendizaje del código Morse. Usted puede
 modificar estas lecciones a su conveniencia.

CW_Player necesita una placa de sonido y Win95 o superior.

Ademas del archivo de Ayuda, cuando el mouse apunta a los controles o 
las opciones se muestran carteles con una breve descripcion.
If you can't open the HLP file under Vista please look at this link :
http://www.microsoft.com/downloads/details.aspx?FamilyId=6EBCFAD9-D3F5-4365-8070-334CD175D4BB&displaylang=en  

Se pueden usar las caracteristicas de Windows como Copiar (Control-C) y 
Pegar (Control-V) series aleatoreas seleccionadas y reproducirlas copiando 
las mismas en el Cuadro de Texto o grabarlar en un archivo usando la opción 
Pegar en su editor favorito. (El Cuadro de Texto solo acepta un registro, 
solo una linea de un archivo ASCII puede ser pegada en este Cuadro)

Este programa ha sido ideado para ser descargado por los navegadores de la 
Web desde los sitios de radio aficionados.
Su proposito es el de iniciar a los no radio aficionados y a los radio 
aficionados ya licenciados sin código, en el código Morse y ayudarlos a 
entrar en las bandas de HF.

Por ello, si usted administra un sitio Web para radio aficionados, no dude en
ofrecer este programa a sus visitantes. 

Ademas, este programa tambien puede ser utilizado para el entrenamiento de 
código Morse, por instructores de CW o para entremamiento en el aire 
conectando una de las salidas de su placa de sonido a la entrada de audio de 
su tranceptor de BLU o FM.

VERY IMPORTANT : 
While running this program under VISTA, WIN7, 8, 10 from a sub-directory under the directory Program Files, 
you will have to start this program as administrator anyway for getting full access to all functions of it, especially the recording of text files, image files, sound files. 
If you don’t do that, the produced files won’t be accessible because of the UserAccountControl (UAC) of Windows.
If you don’t want to run the program as admin, you have to install it in a sub-directory which is NOT under C:\Program Files,
but anyw<here else, for example C:\MyPrograms\. There, the function of the UAC takes no effect.

Version 1.5 :
=============
- más códigos Q

- elija entre idioma Ingles y Frances.

El idioma se puede elegir por línea de comandos, ingresando los siguientes 
parámetros : /e para Ingles, /f para Frances. Si no se ingresa parametro, 
CW_Player inicia en Ingles.
Por ejemplo, para iniciar en Frances, ejecute c:\cw_player.exe /f ,suponiendo 
que el archivo ejecutable se llama cw_player.
CUIDADO que en Win95, cuando se utiliza el comando Ejecutar o se crea un 
acceso directo, la línea de comando se muestra entre comillas.
Entonces, el parametro debe ser ingresado luego de cerradas las comillas :
      "C:\CW_Player.EXE" /f

- Posibilidad de crear sus propios caracteres locales, tales como vocales 
acentuadas. Muchos idiomas tienen este tipo de caracteres locales o 
especiales para los cuales el código Morse convencional no tiene posibildad 
de representarlos.
Esta opción tambien nos permite crear caracteres especiales como SK (fin de 
la transmisión)o AS (esperar).
Estos caracteres locales seran reproducidos en todos los modos del programa.
Cuando se configuran, las caracteristicas de los mismos son guardadas en el 
archivo de configuración CWPLAYER.DAT.

     pantalla principal, pero sin pedir confirmación.

  programa se guarda la sesión de la lección de modo que cuando se vuelve a

- Se corrigio un error del sistema que se podia producir a velocidades de 
reproducción muy bajas.

VERSION 2.0
===========
- Agregado de la opción de Monitoreo la cual realiza el decodificado de las 
señales de Morse ingresadas por el usuario como audio por le entra de
