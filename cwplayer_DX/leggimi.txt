              CW_PLAYER
 ===========================================
   Scritto da  Gabriel Rivat  F6DQM
      http://f6dqm.free.fr
 ===========================================


Questo programma gratuito genera i caratteri standard morse, i codici Q
le abbreviazioni usuali, serie di caratteri casuali, files di testo. Esso
controlla il vostro apprendimento attraverso un test Quiz e vi aiuta ad 
ottenere una buona manipolazione monitorando i vostri segnali CW. 
Manipolando senza trasmettere e verificando la vostra trasmissione sullo 
schermo vi permettera' di uscire in aria con meno stress e piu' fiducia.
Esso genera anche file, di suono Wave (*.wav) dei caratteri Morse, 
che possono essre suonati da qualsiasi software di multimediale.

I caratteri locali possono essere creati da voi stessi.
Vengono proposte 20 lezioni che potrete modificare come più vi piace.

Esso funziona con una scheda audio e Windows XP (minimo).
Oltre l'aiuto in linea, alcuni cenni di aiuto appaiono quando il mouse 
punta sui comandi principali.
If you can't open the HLP file under Vista please look at this link :
http://www.microsoft.com/downloads/details.aspx?FamilyId=6EBCFAD9-D3F5-4365-8070-334CD175D4BB&displaylang=en  

Tutte le funzioni windows come copia con (Control-C) e incolla con 
(Control-V) sono utilizzabili. Una serie di caratteri puo' essere inserita 
nella casella di testo con un'azione combinata di copia-incolla ed 
eseguita, oppure essere incolla ta in un vostro file  con il vostro 
editore di testo preferito per essere eseguita in seguito.
Notate che la finestra di testo puo' contenere solo una sola linea di 
caratteri ASCII terminata con un a capo (Invio).

Questo programma è stato scrito in origine per essere messo a disposizione 
dei navigatori in internet che visitano il nostro sito per radio-amatori.
Esso ha lo scopo di iniziarli alla telegrafia ed aiutarli a raggiungere il
 nostro mondo. Sara' anche utile ai radioamatori che vogliono apprendere 
il codice Morse per accedere alle bande HF.

Se voi gestite un sito WEB, proponetelo ai vostri visitatori.

Inoltre, puo' anche essere utilizzato per il vostro proprio allenamento o
per gli istruttori per l'allenamento in un club radio o anche collegando 
una delle due uscite della vostra scheda audio sull'entrata di modulazione
del vostro trasmettitore FM o SSB.

VERY IMPORTANT : 
While running this program under VISTA, WIN7, 8, 10 from a sub-directory under the directory Program Files, 
you will have to start this program as administrator anyway for getting full access to all functions of it, especially the recording of text files, image files, sound files. 
If you don’t do that, the produced files won’t be accessible because of the UserAccountControl (UAC) of Windows.
If you don’t want to run the program as admin, you have to install it in a sub-directory which is NOT under C:\Program Files,
but anyw<here else, for example C:\MyPrograms\. There, the function of the UAC takes no effect.

Versione 1.5 :
=============

La lingua puo' essere scelta con la linea di comando a mezzo dei parametri:
/e per l'inglese, /f per il francese.
Senza parametri, CW_Player si esprime in inglese.
Per lavorare in en francese, la linea di comando e' del tipo :
   C:\CW_Player.exe /f
ATTENZIONE : sotto Win95, se si utilizza un file batch occorre racchiudere
la linea di comando tra due doppi apici e i parametri devono essere posti 
dopo il secondo doppio apice come segue:
  "C:\CW_Player.exe" /f

 combinazioni create dalla combinazione di più tasti della tastiera.
 Questi caratteri, detti "locali", saranno suonati in tutti i modi se sono
 stati creati. Essi sono salvati nel file specifico CWPLAYER.DAT

Versione 1.9.0
